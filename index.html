About me

My name is Larry Gibbs, Jr. I currently reside in Atlanta, GA, and first dabbled in coding at the beginning of this year. I remember making my first site...which was pretty crappy seeing as it was just a hodgepodge of random pictures and lines of text. Despite this, I'll never forget the feeling I got when I realized what I could created by just using a few lines of code. Ever since I was young, I loved to create things, and in discovering web-development I've found another outlet that I can use to not only be creative, but to express myself in new ways. I look forward to learning as much as I can so that I can not only expand my knowledge, but be able to push the boundaries of what I can do with code.

I majored in Animation from the Savannah College of Art and Design, so I have a strong interest in the artform. When not coding, I'm either watching movies, anime, creating digital art, or coming up with all kinds of story ideas. I recently just self-published my first book, "Song for Avalon", so I'm somewhat of an author. I hope to eventually find ways to combine these hobbies with my growing knowledge of coding to create new and amazing things.

Unplugged

The website for an imaginary music magazine

I would be lying if I said this project wasn't difficult for me. Once again I was tasked with recreating a mockup, but this time it was a multi-page site. On top of that, all of the pages had to be responsive! I enjoyed learning about the @media CSS tag and how it was used to make sure that your site looks correct no matter what device you're viewing it on. Also, I had to get use to using the CSS to change around the different elements on the page depending on the screen size/device. The labeling for the different sections/divs became EXTREMELY important, not just for organizational purposes, but because if something wasn't labeled correctly or taken into account ahead of time, going back to fix, change, or deal with it later down the line was a real headache (which I learned the hard way, sad to say).

The introduction of ems, as well as using percentages instead of pixels for height, width, or space, was a concept that was very interesting to me. I felt it gave the site, as well as the code a bit more flexibility (which, considering it is a responsive site, is no surprise). Also, in the process of making this site, the inspect window in Google Chrome became my best friend. Whenever I was stuck or wanted to try something out, I would use that window to experiment and see what would happen if I were to take certain actions. It really came in handy, as I could see the changes I made in real-time and adjust accordingly before actually implementing the change in the actual code. Also, if something wasn't working correctly, I could fool around in the window to see what was causing the problem. Very handy thing to have at my disposal!

I learned a lot with this project, and I plan on going back and implementing some of those things in my previous projects (especially going back and making them responsive)!

Steven Universe

Website for one of my favorite cartoons...

I wanted to put together a site for one of my favorite cartoons, Steven Universe. I'm pretty excited to get started on it due to the fact that there are some thing I want to try out. On the main page is a simple picture, with a button that says enter. When the button is pressed, I want the screen to give off a flashing effect before switching to the next page. From what I've looked up so far, this can be accomplished using jQuery and a bit of CSS. I also want a character page thats set-up like a character select screen from a fighting game, which I can use the table element in HTML to do. When a character is selected, I want a screen to appear from one side of the screen containing the character's info, while the table is collapsed on the other, which will possibly utilize jQuery or Javascript. There will also be a locations sections with a list of all the locations from the show. When one is chosen, the background will switch to that location while a transparent screen appears on the side of the screen with the location's info. As I stated before, very excited to get started on this and see what new things I learn and accomplish.
